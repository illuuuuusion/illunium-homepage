# illunium-homepage

Illunium homepage built with Nuxt 3, Vue SFC, TypeScript and Tailwind CSS v3.

This project intentionally does not use Nuxt Content, dynamic slug routes or Markdown content files.

## Setup

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Run type checks:

```bash
npm run typecheck
```

Build for production:

```bash
npm run build
```
